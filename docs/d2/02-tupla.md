# Tupla

Para usar a tupla podemos colocar `,` para referenciar um valor do outro.

# Metodo Count
Para contar quantos atributo com mesmo nome está dentro da trupa

```python
dados = [ 'aria', 1, 'google']
dados.count('aria')

# return: 1
 ```
# navegações sequência.

exemplo de navegação: 

```python
dados = [ 'aria', 1, 'google']
dados[1:2]

# return: 1
 ```

# Interavel
exemplo

```python 
dados = [ 'aria', 1, 'google']

for info in dados:
    print('-->', info)

```
